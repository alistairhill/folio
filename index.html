JavaScript Blackjack

April 2015

I had a blast building this Blackjack game, which really confirmed my love for JavaScript. Though inspired by Mike Hall's CSS layout (dealer/player layout and CSS positioned suits on the cards) I built this from scratch using the MVC pattern and vanilla JS. And like my Simon Says project, I also made it multiple ways; first using one single 'dealt' array to capture card objects which were assigned ownership, before settling on separate 'hand' arrays, which added a bit more complexity (err, fun!) to making the game flow.

JavaScript Simon Says

September 2014

Making this simple Simon Says game was an awesome little challenge and super fun to build from scratch. I made two different versions of this; one, using OO JS and jQuery, which was insightful in itself. Then a second using vanilla JS and the MVC pattern, in which I managed to actually reduce the LOC. Eventually I would like to shift more logic into the model, but nevertheless I enjoyed figuring out troublesome scoping issues, including the use of a sweet little function--that adds or removes event listeners--for switching control between Simon and the user.
